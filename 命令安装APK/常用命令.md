### Gradle 编译命令
```bash
# 给 gradlew 添加执行权限 (仅 macOS/Linux)
chmod +x gradlew

# 清理项目
./gradlew clean

# 编译 Debug 版本
./gradlew assembleDebug

# 编译 Release 版本  
./gradlew assembleRelease

# 编译并安装 Debug 版本  （包含 adb install）
./gradlew installDebug

# 完整构建
./gradlew build
```

### ADB 设备管理
```bash
# 查看连接的设备
adb devices

# 重启 ADB 服务
adb kill-server && adb start-server

# 通过 WiFi 连接设备（需要先 USB 连接）
adb tcpip 5555
adb connect 192.168.1.100:5555
```

### APK 安装卸载
```bash
# 安装 APK
adb install ./app/build/outputs/apk/debug/app-debug.apk

# 强制重新安装（覆盖）
adb install -r ./app/build/outputs/apk/debug/app-debug.apk

# 卸载应用
adb uninstall com.nveasy.bledemo

# 卸载但保留数据
adb shell pm uninstall -k com.nveasy.bledemo
```

### 应用控制
```bash
# 启动应用
adb shell am start -n com.nveasy.bledemo/.MainActivity

# 强制停止应用
adb shell am force-stop com.nveasy.bledemo

# 清除应用数据
adb shell pm clear com.nveasy.bledemo

# 查看应用信息
adb shell dumpsys package com.nveasy.bledemo
```

## 📊 日志调试

### 日志查看
```bash
# 实时查看所有日志
adb logcat

# 清空日志并重新开始
adb logcat -c && adb logcat

# 过滤查看应用日志
adb logcat | grep "NvEasy"

# 查看特定标签
adb logcat -s "BluetoothAdapter"

# 保存日志到文件
adb logcat > debug.log

# 查看崩溃日志
adb logcat | grep "AndroidRuntime"
```

### 高级日志过滤
```bash
# 多标签过滤
adb logcat -s "TAG1,TAG2,TAG3"

# 按优先级过滤 (V/D/I/W/E/F)
adb logcat "*:E"  # 只显示错误级别

# 按包名过滤
adb logcat | grep "com.nveasy.bledemo"
```
